## ✨ Возможности

### 🛡️ Многоуровневый обход блокировок
| Тип блокировки | Метод обхода | Эффективность |

- **Фильтрация по доменам**: Поддержка .com, .net, .org и других TLD

## 📥 Установка

### Системные требования
- **ОС**: Windows 10/11 (64-bit)
- **Права**: Администраторские права
- **Диск**: 100 МБ свободного места

### Шаги установки:
1. **Скачайте** последний [релиз](https://github.com/MagilaWEB/unblock-youtube-discord/releases)
2. **Распакуйте** архив в корень диска (`C:\unblock\` или `D:\unblock\`)
3. **Запустите** `engine.exe` от имени администратора

> ⚠️ **ВАЖНО**: Не распаковывайте на Рабочий стол! Это может вызвать проблемы с путями.

## 🚀 Быстрый старт

### Ручной режим работы:
1. Запустите Unblock с правами администратора
2. Перейдите в настройки Unblock
3. Включите ручной режим
4. Перейдите на главную
5. Запустите службу Zapret

### Автоматический режим работы:
1. Запустите Unblock с правами администратора
2. Перейдите в **Настройки**
3. Отключите **ручной режим**
4. Перейдите на **главную**
5. Нажмите кнопку запуска службы Zapret
6. Следуйте подсказкам UI.

## ⚠️ Решение проблем

### 🔴 Антивирусные предупреждения
WinDivert может вызывать ложные срабатывания антивирусов. Это легитимный драйвер для работы с сетевым трафиком. Добавьте в исключение, отключать антивирус не обязательно.

### 🔴 Конфликты с VPN
- Отключите **TUN режим** в настройках VPN
- Проверьте, не блокирует ли VPN локальные подключения

### 🔴 Обход не работает
**Проверьте:**
1. ✅ Путь без пробелов и кириллицы
2. ✅ Установка в корень диска (не на Рабочий стол)
3. ✅ Отключен брандмауэр для тестирования
4. ✅ Правильно выбран режим работы

**Если проблема осталась:**
1. Возьмите одну из конфигураций в папке `configs/strategy`
2. Создайте свой профиль на его основе
3. Если что-то не понятно обратитесь в [обсуждения](https://github.com/MagilaWEB/unblock-youtube-discord/discussions)

### 🔴 Создание собственных конфигураций
Если стандартные профили не работают, изучите документацию:

Zapret: https://github.com/bol-van/zapret2/blob/master/docs/readme.md

Создайте файл в папке `configs/strategy` и поделитесь рабочим профилем в обсуждениях.
